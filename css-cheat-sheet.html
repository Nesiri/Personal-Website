CSS Selectors Cheat Sheet

A complete reference guide to CSS selectors, including basic, attribute, combinator, pseudo-classes, pseudo-elements, and advanced selectors.

1. Basic Selectors

Universal Selector (*): Selects all elements. Use sparingly due to potential performance issues.

Type Selector (e.g., div, p): Targets all elements of a specific type. Ideal for broad styles.

Class Selector (.class-name): Targets elements with a specific class. Reusable and versatile.

ID Selector (#id-name): Targets an element with a specific ID. Use for unique elements.

2. Attribute Selectors

[attribute]: Selects elements with a specific attribute.

[attribute=value]: Selects elements with a specific attribute value.

[attribute~=value]: Selects elements with an attribute containing a specific word in a space-separated list.

[attribute|=value]: Matches elements with an attribute that matches exactly or begins with 'value-'.

[attribute^=value]: Selects elements where an attribute starts with a specific value.

[attribute$=value]: Selects elements where an attribute ends with a specific value.

[attribute*=value]: Selects elements where an attribute contains a specific substring.

3. Combinator Selectors

Descendant Selector (A B): Selects elements inside another element.

Child Selector (A > B): Selects direct child elements only.

Adjacent Sibling (A + B): Selects the immediate sibling following a specified element.

General Sibling (A ~ B): Selects all siblings that follow a specific element.

4. Pseudo-Classes

:first-child: Selects the first child element within a parent.

:last-child: Selects the last child element within a parent.

:nth-child(n): Selects elements based on their position in the parent.

:hover: Styles elements on hover.

:focus: Styles elements when focused (e.g., input fields).

:active: Styles elements when activated, like clicked buttons.

:visited: Styles visited links.

:checked: Targets checked form elements like checkboxes.

5. Pseudo-Elements

::before: Inserts content before the element's content.

::after: Inserts content after the element's content.

::first-line: Applies styles to the first line of an element's content.

::first-letter: Applies styles to the first letter of an element's content.

6. Advanced Selectors

:not(selector): Selects all elements that do not match the selector.

:is(selector): Matches any element within a list of selectors.

:where(selector): Matches elements in a list without adding specificity.

:has(selector): Selects elements containing specified descendants (experimental).

:focus-within: Styles elements if any of its descendants are focused.

:focus-visible: Applies styles to focused elements that should visually indicate focus.

CSS Cheat Sheet

This guide provides descriptions and usage examples for common CSS properties.

Text and Font Properties

color
Sets the color of the text.
color: blue;
font-size
Specifies the size of the font.
font-size: 16px;
font-family
Sets the font type for text.
font-family: Arial, sans-serif;
font-weight
Adjusts the boldness of text.
font-weight: bold;
text-align
Specifies the horizontal alignment of text.
text-align: center;
text-decoration
Defines decorations for text like underline or line-through.
text-decoration: underline;
text-transform
Modifies text capitalization.
text-transform: uppercase;

Background Properties

background-color
Sets the background color.
background-color: #f0f0f0;
background-image
Sets a background image.
background-image: url('image.jpg');
background-repeat
Defines if/how a background image repeats.
background-repeat: no-repeat;
background-position
Sets the starting position of a background image.
background-position: center center;
background-size
Specifies the size of a background image.
background-size: cover;

Box Model Properties

width
Defines the width of an element.
width: 200px;
height
Defines the height of an element.
height: 100px;
padding
Sets the padding inside an element's border.
padding: 10px;
margin
Sets the space outside an element's border.
margin: 20px;
border
Sets the border of an element.
border: 1px solid #000;
border-radius
Defines rounded corners for elements.
border-radius: 5px;
box-shadow
Adds shadow to an element's box.
box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);

Display and Positioning

display
Specifies how an element is displayed.
display: inline-block;
position
Defines the positioning type (static, relative, absolute, fixed, sticky).
position: relative;
top, right, bottom, left
Offsets an element from the respective side when using position (except static).
top: 10px; left: 5px;
float
Moves elements to the left or right, allowing other content to wrap around.
float: left;
clear
Stops elements from wrapping around floated elements.
clear: both;
z-index
Sets the stack order of positioned elements.
z-index: 10;

Flexbox Properties

display: flex
Activates flexbox layout on a container.
display: flex;
justify-content
Aligns items horizontally within a flex container.
justify-content: center;
align-items
Aligns items vertically within a flex container.
align-items: center;
flex-direction
Sets the main axis direction.
flex-direction: row;
flex-wrap
Controls if items wrap to new lines.
flex-wrap: wrap;
align-self
Overrides align-items for individual items.
align-self: flex-end;

Grid Properties

display: grid
Activates grid layout on a container.
display: grid;
grid-template-columns
Defines the number and width of columns in a grid.
grid-template-columns: 1fr 2fr;
grid-template-rows
Defines the number and height of rows in a grid.
grid-template-rows: 100px auto;
gap
Sets space between grid items.
gap: 10px;
grid-column
Specifies the column position of a grid item.
grid-column: 1 / 3;
grid-row
Specifies the row position of a grid item.
grid-row: 1 / 2;
